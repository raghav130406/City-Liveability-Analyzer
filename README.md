# City Liveability Analyzer

A Streamlit-based web app to compare cities based on livability factors like cost of living, safety, healthcare, pollution, and transport.

## Features
- Compare multiple cities
- Adjust weights for each factor
- Get a livability score
- Visualizations (bar charts, radar charts)

## Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib / Plotly

## How to Run
streamlit run app.py
